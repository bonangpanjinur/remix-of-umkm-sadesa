> **Catatan:** Migration SQL belum dijalankan otomatis. Harus di-paste ke **Supabase Dashboard → SQL Editor** dan dijalankan secara manual.

Semua tabel sudah dilengkapi:
- Row Level Security (RLS) policies
- Index untuk query performa tinggi
- Foreign key constraints
- `created_at` / `updated_at` timestamps

## Catatan Teknis Penting

### Yang Harus Dilakukan Sebelum Menggunakan

1. **Jalankan SQL Migration** di Supabase Dashboard:
   - Buka: `supabase/migrations/20260508000000_phase1_pos_saas.sql`
   - Copy semua isi file
   - Paste ke **Supabase → SQL Editor → New Query**
   - Klik **Run**

2. **Akses POS:**
   - Login ke aplikasi
   - Buka halaman **Akun** → klik **Kasir POS**
   - Atau langsung ke URL `/pos`
   - Ikuti wizard setup usaha (nama usaha + outlet pertama)
